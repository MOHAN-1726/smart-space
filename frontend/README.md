# Smart Space - Frontend

React-based user interface for the Smart Space Classroom Management System.

## 🚀 Features
- Interactive Dashboard with Analytics Widgets.
- Integrated Calendar with Drag-and-Drop.
- Student Performance Module.
- Real-time Notifications.
- Responsive design with Tailwind CSS.

## 🛠 Setup & Installation

1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Configure Environment**:
   Create a `.env` file (or use the provided default):
   ```env
   VITE_API_BASE_URL=http://localhost:5000/api
   ```

3. **Run Dev Server**:
   ```bash
   npm run dev
   ```

## 🏗 Build for Production
```bash
npm run build
```
The build artifacts will be in the `dist/` folder, ready for deployment to Vercel or Netlify.
