# Smart Space Classroom Management System

## Tech Stack
Frontend:
- React
- TypeScript
- Vite

Backend:
- Node.js
- Express
- SQLite
- JWT Authentication
- Socket.io

## Features
- Role-based login
- Attendance tracking
- Leave / OD requests
- No-Due management
- Real-time notifications
- Chat system
- Analytics dashboard

## Setup Instructions

Backend:

cd backend
npm install
npm run dev

Frontend:

cd frontend
npm install
npm run dev
